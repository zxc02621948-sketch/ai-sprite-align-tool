# 更新日誌

本專案使用簡潔更新日誌格式，最新版本放在最上方。

## 2026-06-06

### 新增

* 新增 **匯出 PNG + 資料**，可同時下載對齊後 spritesheet PNG 與 JSON 格子資料。
* JSON 會包含欄列數、單格尺寸、總格數、FPS、輸出尺寸，以及每格 `x, y, w, h` 與錨點資訊。
* 新增 **複製 Godot 設定**，方便把欄列、單格尺寸與 FPS 複製到 Godot 匯入流程。
* 新增 **複製 Unity 設定**，方便用 Unity Sprite Editor 的 Grid By Cell Size 切圖。
* 新增與 [遊戲素材去背助手](https://zxc02621948-sketch.github.io/game-asset-bg-remover/) 的連動，可接收去背後的透明 PNG。
* README 新增遊戲引擎匯出說明與工具連動流程。

### 調整

* 調整輸出快取邏輯，修改欄列、對齊方式、邊距、來源框或遮罩後，會清除舊的對齊結果，避免匯出資料過期。
* 保留既有 PNG 匯出流程，同時補上 JSON 與引擎設定輔助。

### 修正

* 修正縮放留白預覽可能因變數名稱錯誤而無法運作的問題。

## 初版

### 新增

* 支援拖放 PNG / JPG / WebP spritesheet。
* 支援設定欄列數、自動切格、Alpha 偵測與動畫預覽。
* 支援主體中心、底部中心、視覺重心等對齊方式。
* 支援逐列對齊與共享命中點。
* 支援來源框編輯、遮罩編輯、手動錨點與自動擴大格子避免裁切。
* 支援匯出修正後 PNG。
